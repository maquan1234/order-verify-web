# 审单核对 · 最简单的共享方式（一个链接大家用）

> 不需要打包、不需要注册任何云平台、不需要联网服务器。只要你的电脑开着，别人就能用。

## 一、给别人用的链接怎么来

1. 双击本目录里的 **`启动并联网.bat`**。
2. 会自动弹出两件事：
   - 一个黑窗口跑着网页服务（别关）；
   - 另一个窗口跑 Cloudflare 隧道，几秒后显示一行：
     `Your quick Tunnel has been created at: https://xxxx.trycloudflare.com`
3. 复制这串 **https 链接**，发到微信群 / 发给同事。
4. 对方打开链接 → 上传销售单 Excel → 自动核对 → 下载结果。

## 二、注意事项

- **你的电脑要一直开着**，且这个黑窗口不能关；关了链接就失效。
- 链接在你**重启电脑 / 重开 bat** 后会变（Cloudflare 免费隧道的特性）。要长期不变，可升级 Cloudflare 保留域名（付费）或用下面的「稳定方案」。
- 多人同时用没问题，小团队够用。

## 三、想要「固定不变」的链接（可选）

- **方案 A（免费但会变）**：就用上面的 bat，每次重开复制新链接发群。
- **方案 B（固定）**：把代码部署到免费云（GitHub 上已有仓库 `maquan1234/order-verify-web`），
  用 Render / Railway / Koyeb 等「从 GitHub 部署」，拿到一个永久域名。需要你注册其中一个账号（一次性）。
- **方案 C（最稳）**：在你电脑上用 `cloudflared tunnel` 配置「保留隧道」，拿到固定子域名（Cloudflare 付费，很便宜）。

## 四、微信群里自动核查（机器人）说明

- **最简单**：把上面的链接发到群里，谁有表谁点开传，当场出结果。不用机器人也能让大家都用上。
- **真要做「群里发文件→自动回结果」的机器人**：
  - 个人微信机器人（itchat/wechaty 之类）**不推荐**——属于对微信的逆向，容易被封号，不适合做生意用。
  - 正规做法是用 **企业微信**：建个企业微信、拉群、加一个群机器人/自建应用，把核查接口接进去，群友发文件就自动回结果。这套需要你注册企业微信并做少量配置，我可以帮你把「接收文件→调用核查→回结果」的逻辑写好。

> 一句话：先双击 `启动并联网.bat` 把链接发出去，大家马上能用；要不要再上企业微信机器人，你定。
