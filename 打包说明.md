# 打包成桌面程序（发给同事直接用）

把"电商审单核对"做成一个 **Windows 桌面程序（.exe）**，发给同事后双击即用，
**不需要联网、不需要装 Python、不需要配置服务器**。

---

## 一、你（制作者）在自己电脑上打包一次

> 只需打包一次，之后把生成的 exe 发给所有人即可。

### 准备工作（仅首次）
1. 确认本机已装 **Python 3.12**（没装去 https://www.python.org/downloads/ 下载，
   安装时 **务必勾选 "Add Python to PATH"**）。
2. 把 `webapp` 这个文件夹复制到你的电脑上（如果已经在就跳过）。

### 一键打包
1. 进入 `webapp` 文件夹。
2. **双击 `打包.bat`**（就是本目录里的那个批处理文件）。
3. 等待命令行跑完（首次需联网装 PyInstaller，约 1–3 分钟）。
4. 看到 `打包完成！` 后，去 `webapp\dist\审单核对工具.exe` 取程序。

> 如果你更喜欢命令行：在 `webapp` 目录执行 `pyinstaller build.spec` 效果一样。

---

## 二、发给同事 & 使用

- 把 **`审单核对工具.exe`** 通过微信 / QQ / U盘 发给同事。
- 同事收到后 **双击打开**：
  - 会自动弹出浏览器进入审单界面；
  - 同时有一个黑色小窗口显示本地地址（关掉它就停止程序）。
- 同事**不需要安装任何东西**，Windows 10/11 直接能跑。

### 货品数据怎么处理？
- 程序首次运行会把内置的 82 条货品复制到用户目录
  `C:\Users\你的用户名\AppData\Roaming\OrderVerify\products.json`。
- 之后你在"货品管理"页面里**新增/修改/删除**的货品都保存在这里，重启不丢。
- 每台电脑各自维护自己的货品表（适合各自业务不同的情况）。
- 想批量同步货品：在货品管理页点"导入"，选一个 Excel 即可覆盖。

## 三、常见问题

**Q：双击 exe 没反应 / 闪退？**
- 看是否杀毒软件拦截，加入白名单即可。
- 打开方式：在 `webapp\dist` 目录里，右键 `审单核对工具.exe` → 以管理员身份运行试试。

**Q：想改端口或换机器保存位置？**
- 一般不用管。程序会自动找一个空闲端口并打开浏览器。
- 高级：可用环境变量 `PORT` 指定端口、`DATA_DIR` 指定数据目录。

**Q：macOS / Linux 同事能用吗？**
- 当前打包的是 Windows 版。如需 Mac 版，在 Mac 上执行同样步骤（需装 Python + 运行 `pyinstaller build.spec`）即可生成 Mac 程序。

**Q：和之前的"网页+隧道"方案冲突吗？**
- 不冲突。本地 exe 是独立方案，完全离线。之前的公网隧道可弃用。

## 四、技术说明（给想了解的人）

- 后端仍是 Flask + openpyxl，前端仍是单页 `templates/index.html`。
- 用 **PyInstaller** 把 Python 解释器、依赖库、模板、货品数据一起打包成单个 exe。
- `app.py` 已做"frozen 感知"：打包后用 `sys._MEIPASS` 定位资源，数据写到用户 `AppData`，
  启动后自动打开浏览器并绑定 `127.0.0.1`，不再依赖任何外部网络。
- 源码已推到 GitHub：`https://github.com/maquan1234/order-verify-web`
